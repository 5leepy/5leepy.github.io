## 1. Pastikan Windows 11 Kamu Sudah Update!

Menjaga sistem operasi tetap terbaru sangat penting, terutama untuk alasan keamanan. Pembaruan juga dapat menambahkan fitur-fitur baru ke PC. Kemungkinan besar PC baru kamu masih menggunakan versi Windows 11 yang lama. Pastikan kamu menggunakan versi Windows 11 terbaru dengan segera memeriksa pembaruan.

Berikut cara memperbarui Windows 11:

1. Buka aplikasi **Settings**.
2. Masuk ke halaman **Windows Update**.
3. Klik **Check for updates**.

![Update Windows](https://5leepy-img.netlify.app/windupdate.jpg)

Pembaruan yang tersedia akan di-download dan diinstal secara otomatis. Beberapa pembaruan mungkin mengharuskan kamu untuk merestart PC. Disarankan untuk memeriksa kembali setelah pembaruan selesai, karena mungkin ada pembaruan tambahan atau beberapa pembaruan gagal diinstal karena konflik.

Selain sistem operasi, kamu juga perlu memperbarui aplikasi bawaan Windows yang ada di Microsoft Store. Caranya:

1. Buka **Microsoft Store**.
2. Klik halaman **Downloads** di pojok kiri bawah.
3. Klik **Get updates** dan tunggu prosesnya selesai. Hampir semua aplikasi bawaan kemungkinan memiliki pembaruan, jadi proses ini mungkin membutuhkan waktu.

![Update Store AppWindows](https://5leepy-img.netlify.app/storeupdate.jpg)

Dengan melakukan pembaruan, kamu akan mendapatkan peningkatan keamanan dan fitur-fitur terbaru yang tersedia.

## 2. Bersihkan Taskbar dan Start Menu

Saat pertama kali menggunakan Windows 11, kamu mungkin akan melihat banyak ikon yang terpasang di taskbar dan Start menu. Beberapa ikon mungkin berguna, namun sebagian lainnya mungkin tidak diperlukan. Sebaiknya bersihkan dan hapus ikon yang tidak kamu perlukan agar tampilannya lebih rapi.
Berikut cara memperbarui Windows 11:

1. Klik kanan pada area kosong di taskbar.
2. Pilih **Taskbar settings**.
3. Di halaman pengaturan, nonaktifkan ikon yang tidak dibutuhkan, seperti Task View, Widgets, dan Search. Kamu juga bisa menyembunyikan ikon aplikasi yang jarang digunakan.

![Windows Taskbar Item Setting](https://5leepy-img.netlify.app/taskbarclean.jpg)

4. Untuk menghapus aplikasi yang terpasang di taskbar secara langsung, cukup klik kanan pada ikonnya dan pilih **Unpin from taskbar**.

![Windows Unpin Taskbar Item](https://5leepy-img.netlify.app/unpintaskbar.jpg)

Untuk Start menu, kamu akan melihat beberapa aplikasi yang terpasang di halaman utama saat membukanya.  Banyak di antaranya mungkin tidak kamu perlukan.  Cukup klik kanan ikon aplikasi yang tidak diinginkan dan pilih **Unpin from Start**. Kamu juga bisa klik **Uninstall** jika kamu ingin menghapus aplikasi tersebut dari PC.

![Windows Unpin Start Item](https://5leepy-img.netlify.app/unpinstart.jpg)

Dengan membersihkan taskbar dan Start menu, tampilan Windows 11 kamu akan menjadi lebih rapi dan mudah digunakan.

## 3. Hapus Aplikasi Bawaan yang Tidak Kamu Butuhkan

PC Windows biasanya dilengkapi dengan banyak aplikasi bawaan. Windows 11 sendiri sudah menyertakan banyak aplikasi, dan jika kamu membeli PC baru, kemungkinan besar produsennya juga telah menginstal banyak aplikasi yang mungkin tidak kamu perlukan. 

Untuk melihat daftar aplikasi yang terinstal:

1. Tekan tombol **`Windows` + `I`** untuk membuka **Settings**.
2. Pilih **Apps** dari sidebar.
3. Buka **Installed Apps**.

Periksa daftar aplikasi dan klik menu tiga titik di samping aplikasi yang ingin kamu hapus. Pilih **Uninstall** dan ikuti instruksi di layar.

![Unistall App Windows](https://5leepy-img.netlify.app/unistallapp.jpg)

>**Perhatian:** Berhati-hatilah saat menghapus aplikasi, karena beberapa produsen menyertakan aplikasi penting seperti  *update tools* untuk driver. Jika kamu tidak yakin apakah suatu aplikasi boleh dihapus, sebaiknya cari tahu lebih lanjut tentang aplikasi tersebut di Google.
{: .prompt-warning }
 
Dengan menghapus aplikasi yang tidak dibutuhkan, kamu dapat menghemat ruang penyimpanan dan sumber daya sistem, sehingga PC kamu dapat berjalan lebih optimal.

## 4. Nonaktifkan Aplikasi Startup yang Tidak Diperlukan

Selain aplikasi bawaan, komputer sering kali dilengkapi dengan aplikasi yang diatur untuk memulai secara otomatis saat sistem booting.  Hal ini dapat memperlambat PC kamu tanpa memberikan banyak manfaat. Sebaiknya periksa daftar aplikasi startup dan nonaktifkan aplikasi yang tidak benar-benar kamu perlukan agar PC kamu bisa booting lebih cepat.

Untuk melihat aplikasi yang berjalan saat PC dinyalakan (dan menonaktifkannya):

1. Klik kanan taskbar dan pilih **Task Manager**.
2. Buka tab **Startup apps** di sisi kiri.

![Task Manager](https://5leepy-img.netlify.app/taskmanagerstart.jpg)

3. Pilih aplikasi yang tidak ingin dijalankan otomatis saat startup, lalu klik **Disable** di bagian atas.

![Disable Start App](https://5leepy-img.netlify.app/disablestart.jpg)

4. Ulangi proses ini untuk aplikasi lain yang ingin kamu nonaktifkan.

>**Tips:** Seperti halnya menghapus aplikasi, pastikan kamu tidak menonaktifkan aplikasi yang benar-benar kamu perlukan untuk menggunakan PC secara normal.  Namun,  jika kamu  menonaktifkan aplikasi yang seharusnya tidak dinonaktifkan, kamu  bisa mengaktifkannya kembali dengan mudah dan merestart PC.
{: .prompt-tip }

Dengan menonaktifkan aplikasi startup yang tidak diperlukan, PC kamu akan booting lebih cepat dan  kinerjanya pun  akan lebih optimal.

## 5. Ubah Tampilan Windows 11

Windows 11 menyediakan beberapa cara untuk menyesuaikan tampilannya agar sesuai dengan gaya kamu.

1. Buka aplikasi **Settings** dan pilih halaman **Personalization** dari menu. Kamu akan melihat semua opsi yang dapat diubah.
2. Di bagian atas halaman, kamu dapat memilih dari beberapa tema preset yang akan mengubah gambar latar belakang dan warna tema. Kamu juga akan melihat opsi individual lainnya yang tercantum di bawah.

![Personalisation Settings](https://5leepy-img.netlify.app/personalisation.jpg)

3. Pilih **Background** untuk mengubah tampilan latar belakang desktop, termasuk mengatur slideshow.
4. Halaman **Colors** memungkinkan kamu memilih antara mode terang atau gelap. Kamu juga dapat memilih warna aksen yang bisa berupa warna apa pun yang kamu inginkan.

![Personalisation Settings](https://5leepy-img.netlify.app/colorpersonal.jpg)

5. Halaman **Themes** mencakup paket yang mengubah kedua pengaturan ini, ditambah penyesuaian untuk suara dan gaya kursor mouse.

>Personalisasi taskbar adalah opsi menarik lainnya. Kamu dapat menyesuaikan orientasi ikon taskbar, yang secara default berada di tengah, kembali ke sudut kiri layar seperti versi Windows sebelumnya.
{: .prompt-info }

Dengan mengubah tampilan Windows 11, kamu dapat membuat pengalaman menggunakan PC menjadi lebih personal dan menyenangkan.

## 6. Aktifkan History Clipboard

Menyalin dan menempel item adalah hal yang biasa kita lakukan. Namun, terbatas hanya pada satu item di clipboard dalam satu waktu bisa sangat menjengkelkan. Windows 11 (dan 10) memiliki solusi untuk ini - kamu sekarang dapat menyimpan banyak item di clipboard dengan mengaktifkan history clipboard.

1. Tekan tombol **`Windows` + `V`** untuk membuka jendela history clipboard.

![Personalisation Settings](https://5leepy-img.netlify.app/clipboardon.png)

2. Fitur ini mungkin dinonaktifkan secara default. Klik **Turn on** untuk mengaktifkannya.
3. Setelah diaktifkan, tekan **`Windows` + `V`** lagi untuk membuka daftar history clipboard dan klik item yang ingin kamu salin.

Setelah kamu memilih item di history clipboard, item tersebut akan menjadi item yang dapat kamu tempel menggunakan **`Ctrl` + `V`** hingga kamu memilih item lain dari daftar atau menyalin sesuatu yang baru. Fungsi ini berguna jika kamu ingin menempelkan sesuatu beberapa kali. 

>History clipboard akan diatur ulang saat kamu merestart atau mematikan komputer. Namun, kamu dapat menyematkan item agar selalu tersedia.
{: .prompt-info }

## 7. Kelola Notifikasi

Secara default, hampir setiap aplikasi yang terinstal di PC kamu diizinkan untuk mengirimkan notifikasi, dan ini mungkin akan terasa mengganggu. Jika kamu ingin mengontrol aplikasi mana saja yang boleh mengirimkan notifikasi dan bagaimana notifikasi tersebut ditampilkan, buka aplikasi **Settings**, navigasikan ke bagian **System**, lalu pilih **Notifications**. 

Di sini, kamu akan melihat pengaturan notifikasi umum beserta semua aplikasi di PC yang dapat mengirimkan notifikasi. Kamu dapat mengubah pengaturan untuk sebagian atau semua aplikasi sesuai keinginanmu.

![Notification Settings](https://5leepy-img.netlify.app/notification-settings.png)

Jika kamu menggulir ke bagian bawah halaman ini, kamu juga akan menemukan **Additional Settings** yang terkait dengan notifikasi Windows dan tips yang seharusnya membantu kamu menggunakan PC. Jika menurutmu ini tidak berguna, kamu dapat menonaktifkannya di sini.

Dengan mengelola notifikasi, kamu dapat meminimalisir gangguan dan fokus pada hal yang lebih penting saat menggunakan PC.

## 8. Nonaktifkan Backup OneDrive

OneDrive adalah layanan penyimpanan cloud dari Microsoft yang terintegrasi dengan Windows 10 dan 11. Secara default, OneDrive diatur untuk mencadangkan folder Desktop, Pictures, dan Documents kamu secara otomatis. Meskipun ini bisa menjadi hal yang baik, kamu mungkin tidak selalu ingin menggunakan penyimpanan cloud. Belum lagi, kamu bisa berakhir dengan banyak duplikat di cloud, karena OneDrive tidak terlalu efisien dalam mencadangkan dari beberapa perangkat.

Jika kamu ingin menonaktifkan backup OneDrive atau mengubah pengaturannya:

1. Klik ikon cloud di dekat pojok kanan bawah taskbar.
2. Klik ikon **roda gigi** di bagian atas flyout, lalu pilih **Settings**.

![OneDrive Taskbar](https://5leepy-img.netlify.app/onedrivesetting.png)

3. Buka tab **Sync and backup**.

![OneDrive Settings](https://5leepy-img.netlify.app/onedrivemanage.png)

4. Klik **Manage backup**.
5. **Matikan** backup untuk folder yang tidak ingin dicadangkan ke OneDrive.

![OneDrive Backup Off](https://5leepy-img.netlify.app/onedrivebackupoff.png)

Jika kamu menggulir lebih jauh ke bawah, kamu juga akan menemukan opsi **Advanced settings** di mana kamu dapat memilih bagaimana kamu ingin mengakses file OneDrive di PC. Kamu dapat mengunduh seluruh library OneDrive agar dapat diakses secara offline, atau menggunakan fitur file on-demand untuk menghemat ruang disk.

## 10. Ganti Browser Default

Browser web bisa dibilang merupakan aplikasi terpenting di PC. Hampir semua yang kita lakukan sekarang berbasis web, jadi ini mungkin aplikasi yang paling kamu butuhkan. Secara default, Windows 11 hadir dengan Microsoft Edge, yang merupakan browser yang hebat. Tetapi jika kamu lebih terbiasa dengan Google Chrome atau yang lainnya, kamu mungkin ingin menggantinya.

Kamu dapat mengunduh browser favoritmu menggunakan tautan di bawah ini:

* [Download Google Chrome](https://www.google.com/chrome/)
* [Download Mozilla Firefox](https://www.mozilla.org/firefox/new/)
* [Download Opera](https://www.opera.com/)
* [Download Brave Browser](https://brave.com/)

Setelah menginstal browser favoritmu, kamu harus mengaturnya sebagai default. Biasanya kamu akan diminta untuk melakukannya segera setelah meluncurkan browser. Jika kamu mengklik tombol untuk mengubah browser default, kamu mungkin akan melihat daftar aplikasi yang terinstal. Cari browser yang kamu inginkan, klik, lalu klik tombol **Set default** di bagian atas halaman. Ini akan mengubah semua asosiasi file dan protokol yang relevan ke browser yang baru saja kamu instal.
